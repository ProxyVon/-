HTML

  • Язык разметки для создания веб-страниц.
  • Состоит из тегов, которые описывают содержимое.
  • Используется в сочетании с CSS и JavaScript.

CSS

  • Язык стилей для оформления веб-страниц.
  • Позволяет управлять внешним видом HTML элементов.
  • Можно использовать для создания адаптивного дизайна.

JavaScript

  • Язык программирования для создания интерактивных веб-страниц.
  • Позволяет работать с элементами HTML и CSS.
  • Используется для создания интерактивного контента.
